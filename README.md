# 📊 Customer Behaviour Analysis – End-to-End Data Analytics Project

## 📌 Project Explanation

This project demonstrates a complete real-world data analytics workflow where I converted raw customer transaction data into meaningful business insights using Python, SQL, and Power BI.

The objective was to understand customer purchasing behaviour, identify revenue patterns, and generate data-driven business recommendations.

## 🔹 Step 1: Data Cleaning & Preparation (Python)

First, I imported the raw dataset into Jupyter Notebook.

I cleaned the data by:

* Handling missing values
* Correcting inconsistent formats
* Standardizing column names and data types
* Removing duplicates

Then, I performed Exploratory Data Analysis (EDA) to understand:

* Revenue distribution
* Customer demographics
* Purchase frequency
* Product category performance

I used **Pandas and NumPy** for data manipulation and basic statistical analysis.

---

## 🔹 Step 2: Database Creation & Business Analysis (SQL)

After cleaning the dataset, I loaded the processed data into a SQL database (MySQL/PostgreSQL/MS SQL Server).

I created structured tables and wrote SQL queries to:

* Segment customers based on spending behaviour
* Identify high-value customers
* Analyze revenue by age group and category
* Evaluate loyalty and subscription impact
* Find key purchase drivers

I used SQL operations such as:

* SELECT
* JOIN
* GROUP BY
* Aggregations (SUM, COUNT, AVG)
* Filtering and sorting

This step helped simulate real business-level data analysis.

---

## 🔹 Step 3: Dashboard Development (Power BI)

Next, I connected Power BI to the SQL database.

I built an interactive dashboard that includes:

* Revenue KPIs
* Customer segmentation charts
* Demographic analysis
* Purchase trend visualization
* Category-wise performance analysis

The dashboard allows users to filter and explore insights dynamically, helping stakeholders make data-driven decisions.

## 🔹 Step 4: Reporting & Business Recommendations

Finally, I summarized the findings in a structured report and presentation.

I highlighted:

* Key customer segments driving revenue
* Trends in purchasing behaviour
* Business improvement opportunities
* Strategic recommendations based on data insights
